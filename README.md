# İnteraktif Medya Kitaplığı

Bu proje, Web Teknolojileri dersi kapsamında geliştirilmiş bir Single Page Application (SPA) medya kütüphanesidir.

## Özellikler
- Film/Dizi Listeleme
- Arama ve Filtreleme
- Detay Görüntüleme
- Favorilere Ekleme (LocalStorage)

## Teknolojiler
- HTML5, CSS3, JavaScript (ES6+)
- Fetch API
- LocalStorage

## Kurulum
Projeyi klonlayın ve `index.html` dosyasını tarayıcıda açın.
